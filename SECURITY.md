# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

Please report the vulnerability as an [issue](https://github.com/jerus-org/mockd/issues/new/choose) or pull [request](https://github.com/jerus-org/mockd/compare).
